# Test Sonuçları Klasörü

Bu klasör, kişiselleştirme sisteminin test sonuçlarını içerir.

## Klasör Yapısı

```
test_sonuclari/
├── README.md                    # Bu dosya
├── rapor_sablonu.md            # Test raporu şablonu
├── test_1_[tarih].md           # Test raporu örnekleri
├── test_2_[tarih].md
└── ...
```

## Test Raporu Oluşturma

1. `rapor_sablonu.md` dosyasını kopyalayın
2. Dosya adını `test_[numara]_[tarih].md` formatında verin
3. Test sonuçlarını doldurun
4. Bu klasöre kaydedin

## Raporlama Standartları

- Her test en az 5 soru içermeli
- Her soru için detaylı değerlendirme yapılmalı
- İstatistikler ve grafikler eklenmeli
- Öneriler ve iyileştirmeler belirtilmeli

## Başarı Kriterleri

- **Mükemmel:** Ortalama puan 20-25 (✅)
- **İyi:** Ortalama puan 15-19 (⚠️)
- **Orta:** Ortalama puan 10-14 (❌)
- **Kötü:** Ortalama puan 0-9 (❌❌)

## Hedef

Sistemin %80+ başarı oranına ulaşması hedeflenmektedir.
